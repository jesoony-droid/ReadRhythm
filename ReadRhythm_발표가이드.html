Q&A 대비 — 예상 질문 & 모범 답변

RAG와 일반 ChatGPT의 차이가 뭔가요?
일반 GPT는 학습 데이터로만 답변하지만, RAG(Retrieval-Augmented Generation)는 실제 데이터베이스에서 관련 정보를 먼저 검색하고 그것을 근거로 답변을 생성합니다. 국립중앙도서관의 실제 사서 Q&A가 근거가 되기 때문에, 도서관 이용·자료 추천·저작권 등에서 훨씬 정확하고 신뢰할 수 있는 답변이 나옵니다.
5,910건 데이터는 어떻게 수집했나요?
국립중앙도서관 홈페이지의 디지털 사서 Q&A 게시판을 Python으로 크롤링했습니다. 질문-답변 쌍을 추출하고, sentence-transformers로 384차원 벡터로 변환해 PostgreSQL+pgvector에 저장했습니다. 임베딩 모델은 로컬에서 실행해 API 비용이 들지 않습니다.
Gemini와 Claude를 둘 다 쓴 이유가 뭔가요?
역할에 맞는 모델을 선택했습니다. Gemini 2.5 Flash는 스트리밍 속도가 빠르고 무료 티어가 있어 RAG 실시간 답변에 적합합니다. Claude Opus 4.7은 Extended Thinking으로 복잡한 추론이 필요한 독서 패턴 분석·추천에 사용했습니다. 각 AI의 강점을 역할에 맞게 배분한 것입니다.
v2.0어떤 개선이 가장 의미 있었나요?
Task #6 컨텍스트 바와 Task #4 서재 담기의 조합이 가장 임팩트가 컸습니다. 읽는 책을 인식 → 제안 칩 자동 생성 → 질문 → 답변에서 바로 담기라는 흐름이 만들어지면서, AI 사서가 단순 Q&A를 넘어 독서 여정 전체의 조력자가 되었습니다.
